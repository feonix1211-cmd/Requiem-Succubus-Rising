## About

**Requiem – Succubus Rising** is a handcrafted Skyrim Anniversary Edition modlist built around one philosophy:

> **Every decision has consequences.**

Unlike traditional Skyrim modlists, this project focuses on immersive roleplaying, meaningful progression, modern combat, and a dark fantasy atmosphere where survival must be earned.

This is **not** a power fantasy.

Combat is brutal.
The world is unforgiving.
Your choices matter.
